## Pre-requisites:
1. Install Node https://nodejs.org/en/download/
2. Install Hugo https://github.com/gohugoio/hugo/releases See expected version in [netlify.toml](./netlify.toml)). Extended version required.

## Installation
1. npm install

## Run
1. npm start
